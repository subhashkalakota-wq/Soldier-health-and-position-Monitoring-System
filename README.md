# Soldier-health-and-position-Monitoring-System
The Soldier Health &amp; Position Tracking System is designed to continuously monitor the vital health parameters of soldiers such as heartbeat and body temperature, while simultaneously tracking their real-time geographical location.  This system is crucial for enhancing soldier safety during missions, enabling command units to react quickly 
🪖 Soldier Health and Position Tracking System

📖 Overview

The Soldier Health and Position Tracking System is an IoT-based solution designed to monitor the real-time health status and location of soldiers in the field. The system continuously collects vital parameters such as heart rate and body temperature and transmits them along with GPS coordinates to a remote monitoring station using GSM communication.

This helps in ensuring soldier safety, quick medical response, and efficient battlefield management.

⸻

🎯 Objectives
	•	Monitor soldier health parameters in real-time
	•	Track live location using GPS
	•	Send emergency alerts automatically
	•	Improve safety and response time in critical situations

⸻

⚙️ Components Used
	•	Microcontroller (Arduino / ESP32)
	•	Heartbeat Sensor
	•	Temperature Sensor
	•	GPS Module
	•	GSM Module (SIM800L)
	•	Power Supply
	•	Connecting wires and breadboard

⸻

🔍 System Architecture

Description:

The system is composed of three main sections:
	1.	Sensing Unit
	•	Collects health data (heart rate, temperature)
	2.	Processing Unit
	•	Microcontroller processes sensor data
	3.	Communication Unit
	•	GPS provides location
	•	GSM sends data to remote system
